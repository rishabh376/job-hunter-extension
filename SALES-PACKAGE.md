### Step 2: Load Extension
1. Open Chrome: `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the extension directory

### Step 3: Configure for Demo
1. Click extension icon ‚Üí "Settings"
